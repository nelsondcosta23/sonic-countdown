# Sonic Speed HUD | Countdown

A high-fidelity countdown timer inspired by the iconic (and traumatic) drowning sequence from Sonic the Hedgehog (1991).

This project transforms the original minimalist progress bar into a High-End cockpit interface, using Glassmorphism aesthetics, dynamic neon glows, and precise audio orchestration to recreate the absolute tension of the Labyrinth Zone.

## The Legacy of Trauma (Sonic 1 PTSD)

If you grew up in the 90s, this script is not just a utility; it's an emotional trigger.

> "They say time heals all wounds, but whoever wrote that never tried to catch an air bubble in the final act of Labyrinth Zone while the music sped up."

This timer was specifically developed to recreate the STO (Sonorous Traumatic Onset) we suffered as children. The official 12-second countdown has been meticulously calibrated to ensure that, upon reaching second 5, you feel exactly the same urge to jump off the couch as you did in 1991.

Note: We are not responsible for sweaty palms or sudden anxiety when hearing the final sound effect.

## Features

- 12-Second Precision: The exact time Sonic can hold his breath.
- Glassmorphism Aesthetics: Translucent panels with background blur and infinite borders.
- Dynamic Stress Levels:
  - Below 10s: The HUD changes to gold (Warning).
  - Below 5s: The HUD enters a pulsing red panic mode (Critical).
- Latency Compensation: 100ms early audio trigger for perfect synchronization with the visual "zero".
- Intuitive Gestures: Drag up/down anywhere on the HUD to calibrate the time.
- Full HUD Mode: Expansion to full screen with a light frame effect (inset glows).

## How to Use

1. Open index.html in any modern browser.
2. Drag the central panel up or down to adjust the initial time.
3. Tap/Click the center to start the challenge.
4. If (or when) you "drown", use the TRY AGAIN button for an instant restart.

## Technologies

- Modern Javascript (ES6): Refactored into classes for maximum performance.
- Advanced CSS3: Backdrop filters, radial gradients, and complex animations.
- Dynamic SVG: Progress ring with real-time circumference calculations.
- Hammer.js: Robust support for gestures and touch interactions.
